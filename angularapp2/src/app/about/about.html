About Reservation Manager

The Reservation Manager is a lightweight, user-friendly application designed to help you manage and organize location-based reservations efficiently.

Whether you're overseeing meeting rooms, equipment bookings, or venue scheduling, this app makes it simple to create, edit, and track reservations all in one place.

Main Features:

  • Create new reservations with start and end times
  • Assign locations and attach optional images
  • Mark reservations as active or inactive
  • View all bookings in a clean, sortable list
  • Edit or update existing reservations with ease

Built with Angular, Bootstrap, and a PHP + MySQL backend, the Reservation Manager is ideal for small-to-medium scale scheduling needs.
